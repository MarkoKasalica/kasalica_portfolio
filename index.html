Recent Projects

Movie App

A web application built in ReactJS. It has a search filter which communicates with The Movie Database API (TMDB). The app fetches data from the metioned API and displays name, image, IMDB rating and short description of the searched movie.

Recipe App

This app works as a book of all kind of food recipes. With similar logic as the MovieApp, it is built in ReactJS and has implemented third-party API. This time I used Edamam API. On the first page we have a input form made for searching recipe. Results sent from Edamam database give us an image of searched food, a title, a recipe and a short description of main nutrients. On the main food card we also have a "See more" button which leads us to programmed modal with details about searched recipe and its nutrient value.

ExpressJS Authentication

This project is a clear exception in comparison with all the others presented here. Main difference lies in fact that this project can't be percived as FrontEnd project. Built with ExpressJS, a framework made for simplifying usage of NodeJS - most popular BackEnd platform written in JavaScript. Here we have an exercise for studying and implementing basic BackEnd concepts and functionalities for the Web: creation of a server, defining routes, usage of middleware functions, usage of cookies, sessions, MongoDB database, hashing of a password and finally defining the register and login process for Authentication of users. We didn't use FrontEnd form so everything was tested with Postman API platform.

Material UI Blog

In this project I used Material UI, one of the most popular libraries of ReactJS UI components that implements Google's Material Design. It made process of styling much easier and faster. The interface is built to look as a blog app with other functionalities, such as using your personal account and chating with other users(friends or connections). It's important to mention that this is just a design without any real functionalities, except for button made for changing the dark/light mode.

ReactJS Expense Tracker

This is a simple ReactJS project for exercising basic concepts of working in React environment. Main feature feature here refers to impementation of input/output logic which implies usage of props, an object that is passed through React components. Process of manipulating the props (short for properties) here starts with gaining the value from user in our form to displaying the entered value (in this case expence) with details like date and price.
